使用者問題如何轉成 Graph 查詢語法

NL Question

e.g. 糖尿病常見症狀？

→

Intent JSON

intent=list_symptoms

→

Template Pick

對應預定 Cypher

→

Param Bind

$disease = Diabetes

→

Execute

rows + answer

{

"intent": "list_symptoms",

"entityType": "Disease",

"entityName": "Diabetes"

}

MATCH (d:Disease {name:$disease})-[:HAS_SYMPTOM]->(s:Symptom)

RETURN d.name AS disease, collect(s.name) AS symptoms

LIMIT 20
