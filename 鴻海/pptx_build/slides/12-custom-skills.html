10. 自訂技能插件開發（SKILL.md）

技能一：system-info

  • 查詢本機時間、uptime、磁碟狀態
  • frontmatter 宣告 requires.bins=[date]
  • 限制長時間網路診斷指令

name: system-info

metadata.openclaw.requires.bins: [date]

date '+%Y-%m-%d %H:%M:%S %Z'

df -h / | tail -1

技能二：file-digest

  • 讀取指定檔案並產生結構化摘要
  • 禁止敏感檔（.env、id_rsa）
  • 大檔案只讀前 200 行防資源濫用

name: file-digest

head -200 /path/to/file

file /path/to/file

python3 -m json.tool data.json
