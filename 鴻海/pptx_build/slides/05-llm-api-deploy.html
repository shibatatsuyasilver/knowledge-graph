LLM API 部署方式（Ollama + FastAPI）

Host / Docker

Ubuntu or macOS

→

Ollama

qwen2.5:7b-instruct

→

FastAPI

/kg/extract /kg/query

→

Neo4j

bolt://localhost:7687

部署關鍵命令（示意）

  • ollama pull qwen2.5:7b-instruct
  • uvicorn main:app --host 0.0.0.0 --port 8000
  • docker run -p 7474:7474 -p 7687:7687 neo4j:5

API 定義

  • GET /health：服務與依賴健康檢查
  • POST /kg/extract：文本抽取 entity/relation
  • POST /kg/query：NL→Intent→Cypher→Answer
